在 MySQL 中，对数据的操作主要围绕 **CRUD**（创建、读取、更新、删除）展开，以下是常用的操作命令及示例：


### 1. 插入数据（Create）
使用 `INSERT INTO` 语句向表中添加新记录。

#### 基本语法：
```sql
-- 插入指定字段
INSERT INTO 表名 (字段1, 字段2, ...) VALUES (值1, 值2, ...);

-- 插入所有字段（需按表中字段顺序）
INSERT INTO 表名 VALUES (值1, 值2, ...);

-- 批量插入
INSERT INTO 表名 (字段1, 字段2) VALUES (值1, 值2), (值3, 值4), ...;
```

#### 示例：
假设有表 `users`（id, name, age）：
```sql
-- 插入单条记录
INSERT INTO users (name, age) VALUES ('张三', 25);

-- 批量插入
INSERT INTO users (name, age) VALUES 
  ('李四', 30),
  ('王五', 28);
```


### 2. 查询数据（Read）
使用 `SELECT` 语句从表中读取数据，是最常用的操作之一。

#### 基本语法：
```sql
-- 查询所有字段
SELECT * FROM 表名;

-- 查询指定字段
SELECT 字段1, 字段2 FROM 表名;

-- 带条件查询（WHERE）
SELECT 字段 FROM 表名 WHERE 条件;

-- 排序（ORDER BY，ASC升序/ DESC降序，默认升序）
SELECT 字段 FROM 表名 ORDER BY 字段 [ASC/DESC];

-- 限制查询结果条数（LIMIT）
SELECT 字段 FROM 表名 LIMIT 数量;

-- 去重（DISTINCT）
SELECT DISTINCT 字段 FROM 表名;
```

#### 示例：
```sql
-- 查询所有用户
SELECT * FROM users;

-- 查询年龄大于25的用户姓名和年龄
SELECT name, age FROM users WHERE age > 25;

-- 按年龄降序排列，取前2条
SELECT name, age FROM users ORDER BY age DESC LIMIT 2;
```


### 3. 更新数据（Update）
使用 `UPDATE` 语句修改表中已有记录，**务必加 WHERE 条件**，否则会更新所有记录。

#### 基本语法：
```sql
UPDATE 表名 SET 字段1=值1, 字段2=值2 WHERE 条件;
```

#### 示例：
```sql
-- 将张三的年龄改为26
UPDATE users SET age = 26 WHERE name = '张三';

-- 将所有年龄大于30的用户年龄加1
UPDATE users SET age = age + 1 WHERE age > 30;
```


### 4. 删除数据（Delete）
使用 `DELETE` 语句删除表中记录，**务必加 WHERE 条件**，否则会删除所有记录。

#### 基本语法：
```sql
DELETE FROM 表名 WHERE 条件;
```

#### 示例：
```sql
-- 删除名字为李四的用户
DELETE FROM users WHERE name = '李四';

-- 删除年龄小于18的用户
DELETE FROM users WHERE age < 18;
```


### 其他常用操作
- **条件运算符**：`=`（等于）、`!=`/`<>`（不等于）、`>`、`<`、`>=`、`<=`、`BETWEEN ... AND ...`（范围）、`IN`（包含）、`LIKE`（模糊匹配，`%` 通配任意字符，`_` 通配单个字符）、`IS NULL`（空值）等。
  ```sql
  -- 模糊查询：名字包含“张”的用户
  SELECT * FROM users WHERE name LIKE '%张%';
  ```

- **聚合函数**：`COUNT()`（计数）、`SUM()`（求和）、`AVG()`（平均值）、`MAX()`（最大值）、`MIN()`（最小值）。
  ```sql
  -- 统计用户总数
  SELECT COUNT(*) FROM users;
  
  -- 计算平均年龄
  SELECT AVG(age) FROM users;
  ```

- **分组查询**：`GROUP BY` 结合聚合函数使用。
  ```sql
  -- 按年龄分组，统计每个年龄的用户数量
  SELECT age, COUNT(*) FROM users GROUP BY age;
  ```


操作时需注意：
- 执行 `UPDATE` 或 `DELETE` 前，建议先用 `SELECT` 验证条件是否正确，避免误操作。
- 字符串值需用单引号 `''` 包裹，数值类型直接写。
- 关键字不区分大小写，但通常大写以区分 SQL 语法和表/字段名。
