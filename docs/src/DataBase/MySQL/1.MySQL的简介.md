# MySQL 简介
MySQL 是一款开源的关系型数据库管理系统（RDBMS），广泛应用于各类Web应用和数据存储场景，以其高效、稳定和易用性成为主流数据库之一。

## 1. 核心特性
- **开源免费**：基于 GNU 通用公共许可证，可免费下载、使用和修改，降低企业成本。
- **跨平台兼容**：支持 Windows、Linux、macOS 等多种操作系统，适配不同部署环境。
- **高性能**：优化了查询速度，支持索引、分区表等功能，能高效处理大量数据请求。
- **支持多语言**：可通过 Python、Java、PHP、C++ 等多种编程语言进行连接和操作。
- **安全性**：提供用户权限管理、数据加密、SSL 连接等功能，保障数据安全。

---

## 2. 主要应用场景
- **Web 开发**：是 LAMP（Linux + Apache + MySQL + PHP）、LNMP 等经典Web架构的核心组件，用于存储网站用户数据、内容信息等。
- **中小企业数据管理**：适合存储企业客户信息、订单数据、财务报表等结构化数据，维护成本低。
- **嵌入式系统**：体积小巧，可嵌入到手机、机顶盒等设备中，存储设备配置、用户偏好等数据。
- **数据分析辅助**：可作为中小型数据分析项目的数据源，配合 Excel、Python 等工具进行数据处理。

## 4. 基本操作示例
1. **连接数据库**：`mysql -u 用户名 -p`，输入密码后进入 MySQL 命令行。
2. **创建数据库**：`CREATE DATABASE 数据库名;`。
3. **创建表**：在指定数据库中，通过 `CREATE TABLE 表名 (字段1 类型, 字段2 类型);` 定义数据结构。
4. **数据操作**：使用 `INSERT`（插入）、`SELECT`（查询）、`UPDATE`（更新）、`DELETE`（删除）语句处理数据。
