在 Linux 系统中，处理文件和目录的命令是日常操作的基础，以下是常用命令及示例：

`pwd`：查看当前路径

### 1. 查看目录内容


```bash
ls [选项] [目录]
```

+ 常用选项：
    - `-l`：详细列表形式显示（包含权限、大小等信息）
    - `-a`：显示所有文件（包括隐藏文件，以 `.` 开头）
    - `-h`：以人类可读的单位显示大小（如 K、M）
    - `-R`：递归显示子目录内容

示例：



```bash
ls -lha /home  # 详细显示/home目录所有文件（含隐藏）并以友好单位显示大小
```

### 2. 切换目录


```bash
cd [目录路径]
```

+ 特殊用法：
    - `cd` 或 `cd ~`：切换到当前用户的家目录
    - `cd ..`：切换到上级目录
    - `cd -`：切换到上一次所在目录

示例：



```bash
cd /var/log  # 切换到/var/log目录
cd ../tmp    # 从当前目录切换到上级目录的tmp子目录
```

### 3. 创建目录


```bash
mkdir [选项] 目录名
```

+ 常用选项：
    - `-p`：递归创建多级目录（父目录不存在时自动创建）

示例：



```bash
mkdir docs                  # 创建单个目录docs
mkdir -p project/{src,lib}  # 同时创建project目录及其子目录src和lib
```

### 4. 创建空文件


```bash
touch 文件名
```

+ 若文件不存在则创建空文件，若已存在则更新其修改时间

示例：



```bash
touch note.txt          # 创建空文件note.txt
touch logs/{a,b}.log    # 同时创建logs目录下的a.log和b.log
```

### 5. 复制文件 / 目录


```bash
cp [选项] 源文件/目录 目标路径
```

+ 常用选项：
    - `-r` 或 `-R`：递归复制目录（必须用于目录复制）
    - `-i`：覆盖前提示确认
    - `-v`：显示复制过程

示例：



```bash
cp file.txt /backup              # 复制file.txt到/backup目录
cp -r docs/ /tmp                 # 递归复制docs目录到/tmp
cp -iv *.pdf ~/documents         # 复制所有pdf文件到documents目录，覆盖前提示
```

### 6. 移动 / 重命名文件 / 目录


```bash
mv [选项] 源文件/目录 目标路径
```

+ 功能：移动文件 / 目录，或在同一目录下重命名
+ 常用选项：`-i`（覆盖前提示）、`-v`（显示移动过程）

示例：



```bash
mv old.txt new.txt               # 重命名文件
mv report.pdf ~/downloads        # 移动文件到downloads目录
mv -i data/ /archive             # 移动目录到archive，覆盖前提示
```

### 7. 删除文件 / 目录


```bash
rm [选项] 文件/目录
```

+ 常用选项：
    - `-r` 或 `-R`：递归删除目录（必须用于目录删除）
    - `-f`：强制删除（不提示确认）
    - `-i`：删除前提示确认（默认对重要文件生效）

示例：



```bash
rm temp.txt              # 删除单个文件（可能提示确认）
rm -rf old_dir/          # 强制递归删除目录及其内容（慎用！）
rm -i *.tmp              # 删除所有tmp文件，每个都提示确认
```

### 8. 查看文件内容
+ `cat 文件名`：一次性显示整个文件内容（适合短文件）
+ `more 文件名`：分页显示（按空格键翻页，q 退出）
+ `less 文件名`：更强大的分页查看（支持上下滚动、搜索）
+ `head -n 行数 文件名`：显示文件前 N 行（默认前 10 行）
+ `tail -n 行数 文件名`：显示文件后 N 行（默认后 10 行）
+ `tail -f 文件名`：实时跟踪文件新增内容（常用于日志监控）

示例：



```bash
cat config.ini                  # 查看配置文件
less /var/log/messages          # 分页查看系统日志
tail -n 20 access.log           # 查看访问日志最后20行
tail -f /var/log/nginx/error.log # 实时监控nginx错误日志
```

### 9. 搜索文件 / 目录


```bash
find 搜索路径 [选项] 条件
```

示例：



```bash
find /home -name "*.txt"        # 在/home下搜索所有txt文件
find /var -type d -name "log*"  # 在/var下搜索以log开头的目录
find / -size +100M              # 在根目录下搜索大于100M的文件
```

### 10. 查看当前目录路径


```bash
pwd
```

+ 显示当前工作目录的绝对路径

这些命令是 Linux 文件系统操作的核心，熟练掌握后能极大提高工作效率。使用时注意：删除操作（`rm -rf`）需格外谨慎，避免误删重要文件；处理目录时记得添加 `-r` 选项。
