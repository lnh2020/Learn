在 Linux 系统中，系统工作命令用于监控和管理系统状态、进程、资源等，以下是常用的系统工作命令：

## 1. 查看系统信息
+ **uname**：显示系统内核信息
```bash
uname -a  # 显示完整系统信息（内核版本、主机名、硬件架构等）
uname -r  # 仅显示内核版本
```

+ **hostname**：查看或设置主机名
```bash
hostname          # 显示当前主机名
hostname newname  # 临时设置主机名（重启失效）
```

+ **cat /etc/os-release**：查看操作系统版本信息
```bash
cat /etc/os-release  # 显示发行版名称、版本等（适用于主流Linux发行版）
```

## 2. 进程管理
+ **ps**：查看当前进程状态
```bash
ps aux       # 显示所有进程（详细信息，包括用户、CPU占用等）
ps -ef       # 以全格式显示进程（包括父进程ID）
ps aux | grep "nginx"  # 过滤出包含nginx的进程
```

+ **top**：实时监控进程资源占用（动态更新）
```bash
top  # 进入实时监控界面，按 q 退出
# 常用快捷键：P（按CPU排序）、M（按内存排序）、k（终止进程）
```

+ **htop**：`top` 的增强版（需安装，界面更友好）
```bash
htop  # 支持鼠标操作、颜色区分，功能更丰富
```

+ **kill**：终止进程
```bash
kill 进程ID        # 发送终止信号（默认15）给进程
kill -9 进程ID     # 强制终止进程（信号9，无法忽略）
killall 进程名     # 终止所有同名进程（如 killall nginx）
```

## 3. 资源监控
+ **free**：查看内存使用情况
```bash
free -h  # 以人类可读单位（G、M）显示内存和交换分区使用情况
```

+ **df**：查看磁盘空间使用情况
```bash
df -h    # 显示所有文件系统的磁盘使用情况（含挂载点）
df -i    # 显示inode使用情况（避免因inode耗尽导致无法创建文件）
```

+ **du**：查看目录 / 文件的磁盘占用
```bash
du -sh 目录名  # 显示目录总大小（-s：汇总，-h：友好单位）
du -h --max-depth=1  # 显示当前目录下一级子目录的大小
```

+ **vmstat**：监控系统整体资源（CPU、内存、IO 等）
```bash
vmstat 2 5  # 每2秒输出一次，共输出5次（查看系统瓶颈常用）
```

+ **iostat**：监控磁盘 IO 性能（需安装 `sysstat` 包）
```bash
iostat -x 1  # 每1秒显示一次磁盘IO详细统计（-x：扩展信息）
```

## 4. 网络管理
+ **ifconfig**：查看或配置网络接口（传统命令）
```bash
ifconfig        # 显示所有网络接口信息（IP、MAC等）
ifconfig eth0   # 只显示eth0接口信息
```

+ **ip**：新一代网络管理命令（替代 `ifconfig`）
```bash
ip addr show       # 显示所有网络接口的IP地址
ip link set eth0 up/down  # 启用/禁用eth0接口
```

+ **ping**：测试网络连通性
```bash
ping www.baidu.com  # 向目标发送ICMP包，按Ctrl+C停止
ping -c 4 192.168.1.1  # 只发送4个包
```

+ **netstat**：查看网络连接、端口监听等（常用 `-tuln`）
```bash
netstat -tuln  # 显示所有监听的TCP/UDP端口（t：TCP，u：UDP，l：监听，n：数字显示）
netstat -an    # 显示所有网络连接（包括已建立的连接）
```

+ **ss**：`netstat` 的替代工具（更快、功能更多）
```bash
ss -tuln  # 与 netstat -tuln 功能相同，但效率更高
```

## 5. 系统负载与时间
+ **uptime**：显示系统运行时间和负载
```bash
uptime  # 输出格式：当前时间 运行时间 登录用户数 1/5/15分钟平均负载
```

+ **w**：显示当前登录用户及他们的活动
```bash
w  # 显示用户登录时间、运行的命令等
```

+ **date**：查看或设置系统时间
```bash
date                # 显示当前时间
date "+%Y-%m-%d %H:%M:%S"  # 自定义时间格式（年-月-日 时:分:秒）
```

+ **timedatectl**：管理系统时间和时区（systemd 系统）
```bash
timedatectl               # 显示当前时间、时区等信息
timedatectl set-timezone Asia/Shanghai  # 设置时区为上海
```

## 6. 其他实用命令
+ **whoami**：显示当前登录用户名
```bash
whoami  # 快速查看当前用户身份
```

+ **last**：查看用户登录历史
```bash
last  # 显示最近的用户登录记录（包括登录时间、IP等）
```

+ **dmesg**：查看系统启动日志（硬件相关信息）
```bash
dmesg | less  # 查看内核启动过程中的信息（如硬件检测、驱动加载等）
```

+ **systemctl**：管理系统服务（systemd 系统，替代 `service`）
```bash
systemctl status nginx  # 查看nginx服务状态
systemctl start nginx   # 启动服务
systemctl enable nginx  # 设置服务开机自启
```

这些命令是系统管理和问题排查的基础，熟练掌握可以快速了解系统状态、定位资源瓶颈或解决简单的系统问题。
