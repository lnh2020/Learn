在 CentOS 系统中，用户组管理与用户管理紧密相关，以下是常用的用户组管理命令：

## 创建用户组

```bash
groupadd [选项] 组名
```

常用选项：
- `-g`：指定组 ID（GID），不指定则自动分配

示例：创建一个名为 develop 的用户组
```bash
groupadd develop
```

## 修改用户组信息

```bash
groupmod [选项] 组名
```

常用选项：
- `-n`：修改组名
- `-g`：修改组 ID（GID）

示例：将组名 develop 改为 dev
```bash
groupmod -n dev develop
```

## 删除用户组

```bash
groupdel 组名
```

注意：被删除的组不能是任何用户的主组，否则会删除失败。
示例：删除名为 testgroup 的用户组
```bash
groupdel testgroup
```

## 将用户添加到指定组

```bash
usermod -aG 组名 用户名
```

选项说明：
- `-a`：表示追加（append），避免覆盖用户原有的附加组
- `-G`：指定要添加到的附加组

示例：将用户 john 添加到 dev 组
```bash
usermod -aG dev john
```

## 查看用户所属组

```bash
groups 用户名
```

示例：查看用户 john 所属的组
```bash
groups john
```

## 查看系统中所有组

```bash
cat /etc/group
```

该文件存储了系统中所有组的信息，格式为：`组名:密码:GID:成员列表`

## 更改用户的主组

```bash
usermod -g 新主组名 用户名
```

示例：将用户 john 的主组改为 dev
```bash
usermod -g dev john
```

这些命令同样需要管理员权限执行，合理的用户组管理可以简化权限控制，提高系统安全性。例如，将多个用户添加到同一个组中，可以方便地对该组分配文件或目录的访问权限。
