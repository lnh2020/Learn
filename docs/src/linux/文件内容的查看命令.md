<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">在 Linux 系统中，查看文件内容的命令有很多，适用于不同场景（如文件大小、查看方式、实时监控等）。以下是常用的文件内容查看命令：</font>

### <font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">1.</font><font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);"> </font>`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">cat</font>`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);"> </font><font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">- 一次性显示整个文件内容</font>
<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">适合查看</font>**<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">小型文件</font>**<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">，直接输出全部内容到终端。</font>

**<font style="color:rgb(249, 250, 251);background-color:rgb(252, 252, 252);">bash</font>**

```bash
cat 文件名          # 显示文件全部内容
cat -n 文件名       # 显示内容并显示行号
cat -b 文件名       # 显示内容并显示行号（忽略空行）
cat file1 file2     # 同时显示多个文件内容
```

<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">示例：</font>

**<font style="color:rgb(249, 250, 251);background-color:rgb(252, 252, 252);">bash</font>**

```bash
cat /etc/hosts      # 查看主机配置文件
cat -n script.sh    # 显示脚本内容并带行号
```

### <font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">2.</font><font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);"> </font>`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">more</font>`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);"> </font><font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">- 分页查看文件（向前翻页受限）</font>
<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">适合查看</font>**<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">较长文件</font>**<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">，支持按页浏览（空格键翻页，</font>`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">Enter</font>`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);"> </font><font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">键逐行滚动）。</font>

**<font style="color:rgb(249, 250, 251);background-color:rgb(252, 252, 252);">bash</font>**

```bash
more 文件名
```

<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">操作键：</font>

+ <font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">空格键：显示下一页</font>
+ `<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">Enter</font>`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">：显示下一行</font>
+ `<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">b</font>`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">：回退上一页（部分版本支持）</font>
+ `<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">q</font>`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">：退出查看</font>

<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">示例：</font>

**<font style="color:rgb(249, 250, 251);background-color:rgb(252, 252, 252);">bash</font>**

```bash
more /var/log/messages  # 分页查看系统日志
```

### <font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">3.</font><font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);"> </font>`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">less</font>`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);"> </font><font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">- 更强大的分页查看工具</font>
`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">more</font>`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);"> </font><font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">的增强版，支持向前 / 向后翻页、搜索等功能，适合查看</font>**<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">大文件</font>**<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">。</font>

**<font style="color:rgb(249, 250, 251);background-color:rgb(252, 252, 252);">bash</font>**

```bash
less 文件名
```

<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">常用操作：</font>

+ <font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">上下方向键：逐行滚动</font>
+ `<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">PageUp</font>`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">/</font>`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">PageDown</font>`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">：翻页</font>
+ `<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">/关键词</font>`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">：向下搜索关键词（按</font><font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);"> </font>`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">n</font>`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);"> </font><font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">找下一个，</font>`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">N</font>`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);"> </font><font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">找上一个）</font>
+ `<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">?关键词</font>`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">：向上搜索关键词</font>
+ `<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">q</font>`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">：退出</font>

<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">示例：</font>

**<font style="color:rgb(249, 250, 251);background-color:rgb(252, 252, 252);">bash</font>**

```bash
less /etc/passwd  # 查看用户信息文件，支持搜索用户名
```

### <font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">4.</font><font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);"> </font>`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">head</font>`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);"> </font><font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">- 查看文件开头内容</font>
<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">默认显示文件前 10 行，可指定行数。</font>

**<font style="color:rgb(249, 250, 251);background-color:rgb(252, 252, 252);">bash</font>**

```bash
head 文件名             # 显示前10行
head -n 行数 文件名     # 显示前N行
head -c 字节数 文件名   # 显示前N个字节
```

<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">示例：</font>

**<font style="color:rgb(249, 250, 251);background-color:rgb(252, 252, 252);">bash</font>**

```bash
head -n 5 /etc/profile  # 查看环境变量配置文件的前5行
head -c 100 data.bin    # 查看二进制文件的前100个字节
```

### <font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">5.</font><font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);"> </font>`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">tail</font>`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);"> </font><font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">- 查看文件末尾内容</font>
<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">默认显示文件后 10 行，常用于查看日志文件，支持实时监控新增内容。</font>

**<font style="color:rgb(249, 250, 251);background-color:rgb(252, 252, 252);">bash</font>**

```bash
tail 文件名             # 显示后10行
tail -n 行数 文件名     # 显示后N行
tail -f 文件名          # 实时跟踪文件新增内容（按Ctrl+C退出）
tail -F 文件名          # 与-f类似，但文件被删除重建后仍能继续跟踪
```

<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">示例：</font>

**<font style="color:rgb(249, 250, 251);background-color:rgb(252, 252, 252);">bash</font>**

```bash
tail -n 20 /var/log/nginx/access.log  # 查看nginx访问日志最后20行
tail -f /var/log/syslog               # 实时监控系统日志新增内容
```

### <font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">6.</font><font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);"> </font>`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">nl</font>`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);"> </font><font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">- 显示内容并添加行号</font>
<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">与</font><font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);"> </font>`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">cat -n</font>`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);"> </font><font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">类似，但行号格式更灵活（默认忽略空行）。</font>

**<font style="color:rgb(249, 250, 251);background-color:rgb(252, 252, 252);">bash</font>**

```bash
nl 文件名              # 显示内容并添加行号（忽略空行）
nl -ba 文件名          # 显示内容并为所有行（包括空行）添加行号
```

<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">示例：</font>

**<font style="color:rgb(249, 250, 251);background-color:rgb(252, 252, 252);">bash</font>**

```bash
nl script.py  # 查看Python脚本并显示行号
```

### <font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">7.</font><font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);"> </font>`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">tac</font>`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);"> </font><font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">- 反向显示文件内容</font>
<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">与</font><font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);"> </font>`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">cat</font>`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);"> </font><font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">相反，从最后一行开始显示到第一行。</font>

**<font style="color:rgb(249, 250, 251);background-color:rgb(252, 252, 252);">bash</font>**

```bash
tac 文件名
```

<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">示例：</font>

**<font style="color:rgb(249, 250, 251);background-color:rgb(252, 252, 252);">bash</font>**

```bash
tac /var/log/dmesg  # 反向查看系统启动日志（最新内容在前）
```

### <font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">8.</font><font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);"> </font>`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">od</font>`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);"> </font><font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">- 查看二进制文件</font>
<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">以</font>**<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">八进制、十六进制</font>**<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">等格式查看二进制文件（如可执行文件、图片等）。</font>

**<font style="color:rgb(249, 250, 251);background-color:rgb(252, 252, 252);">bash</font>**

```bash
od -t x1 文件名  # 以十六进制单个字节显示
od -t d2 文件名  # 以十进制双字节显示
```

<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">示例：</font>

**<font style="color:rgb(249, 250, 251);background-color:rgb(252, 252, 252);">bash</font>**

```bash
od -t x1 /bin/ls  # 以十六进制查看ls命令的二进制内容
```

### <font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">选择建议：</font>
+ <font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">小文件：直接用</font><font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);"> </font>`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">cat</font>`
+ <font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">大文件分页浏览：用</font><font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);"> </font>`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">less</font>`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">（功能最全）</font>
+ <font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">只看开头 / 末尾：用</font><font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);"> </font>`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">head</font>`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);"> </font><font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">或</font><font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);"> </font>`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">tail</font>`
+ <font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">实时监控日志：用</font><font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);"> </font>`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">tail -f</font>`
+ <font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">二进制文件：用</font><font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);"> </font>`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">od</font>`

<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">这些命令是日常查看文件内容的基础，结合管道（</font>`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">|</font>`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">）还能实现更复杂的操作，例如 </font>`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">cat largefile.txt | grep "error"</font>`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);"> 筛选包含 "error" 的行。</font>
