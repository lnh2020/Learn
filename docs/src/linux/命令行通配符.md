<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">在 Linux 命令行中，通配符（Wildcards）用于匹配文件名或路径，简化文件批量操作。常用的通配符及其用法如下：</font>

### <font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">1.</font><font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);"> </font>`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">*</font>`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">（星号）</font>
+ **<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">功能</font>**<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">：匹配任意数量的任意字符（包括零个字符）</font>
+ **<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">示例</font>**<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">：</font>**<font style="color:rgb(249, 250, 251);background-color:rgb(252, 252, 252);">bash</font>**

```bash
# 列出当前目录下所有 .txt 文件
ls *.txt

# 列出所有以 "log" 开头的文件
ls log*

# 复制所有以 "data_" 开头且以 ".csv" 结尾的文件到 backup 目录
cp data_*.csv backup/
```

### <font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">2.</font><font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);"> </font>`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">?</font>`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">（问号）</font>
+ **<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">功能</font>**<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">：匹配</font>**<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">单个</font>**<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">任意字符</font>
+ **<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">示例</font>**<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">：</font>**<font style="color:rgb(249, 250, 251);background-color:rgb(252, 252, 252);">bash</font>**

```bash
# 匹配文件名长度为3个字符的文件（如 abc、123 等）
ls ???

# 匹配以 "file_" 开头，后面跟1个任意字符，再以 ".txt" 结尾的文件
# 如 file_a.txt、file_1.txt 等
ls file_?.txt
```

### <font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">3.</font><font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);"> </font>`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">[]</font>`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">（方括号）</font>
+ **<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">功能</font>**<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">：匹配括号内的</font>**<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">任意一个字符</font>**<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">（字符集）</font>
+ **<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">用法</font>**<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">：</font>
    - `<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">[abc]</font>`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">：匹配 a、b、c 中的任意一个</font>
    - `<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">[0-9]</font>`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">：匹配 0-9 中的任意一个数字</font>
    - `<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">[a-z]</font>`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">：匹配小写字母 a-z 中的任意一个</font>
    - `<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">[^abc]</font>`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);"> </font><font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">或</font><font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);"> </font>`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">[!abc]</font>`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">：匹配</font>**<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">不在</font>**<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">括号内的任意一个字符（取反）</font>
+ **<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">示例</font>**<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">：</font>**<font style="color:rgb(249, 250, 251);background-color:rgb(252, 252, 252);">bash</font>**

```bash
# 列出所有以 a、b、c 开头的 .sh 文件
ls [abc]*.sh

# 列出文件名包含数字的文件
ls *[0-9]*

# 列出不以 .log 结尾的文件（取反）
ls *[!.]log
```

### <font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">4.</font><font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);"> </font>`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">{}</font>`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">（花括号）</font>
+ **<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">功能</font>**<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">：生成</font>**<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">多个字符串组合</font>**<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">（不是匹配已有文件，而是生成新的模式）</font>
+ **<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">示例</font>**<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">：</font>**<font style="color:rgb(249, 250, 251);background-color:rgb(252, 252, 252);">bash</font>**

```bash
# 创建多个文件（file1.txt、file2.txt、file3.txt）
touch file{1,2,3}.txt

# 复制 file.txt 到 dir1、dir2、dir3 目录
cp file.txt {dir1,dir2,dir3}/

# 生成连续序列（如 1-5 或 a-c）
mkdir dir{1..5}       # 创建 dir1 到 dir5 目录
touch {a..d}.md       # 创建 a.md 到 d.md 文件
```

### <font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">5. 通配符组合使用</font>
<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">通配符可以组合使用，实现更复杂的匹配：</font>

**<font style="color:rgb(249, 250, 251);background-color:rgb(252, 252, 252);">bash</font>**

```bash
# 匹配以 "img_" 开头，中间是2个数字，结尾是 .jpg 或 .png 的文件
ls img_[0-9][0-9].{jpg,png}

# 复制所有 .txt 和 .pdf 文件到 archive 目录
cp *.{txt,pdf} archive/
```

### <font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">注意事项</font>
+ <font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">通配符由 Shell（如 bash）解析，而非命令本身，因此适用于所有接受文件名作为参数的命令</font>
+ `<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">*</font>`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);"> </font><font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">和</font><font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);"> </font>`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">?</font>`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);"> </font><font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">是 "匹配已有文件"，</font>`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">{}</font>`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);"> </font><font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">是 "生成模式"（即使文件不存在也会生成对应的字符串）</font>
+ <font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">某些命令（如</font><font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);"> </font>`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">grep</font>`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">）中使用的</font><font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);"> </font>`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">*</font>`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);"> </font><font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">或</font><font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);"> </font>`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">?</font>`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);"> </font><font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">属于正则表达式，与 Shell 通配符含义不同，需注意区分</font>

<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">通配符是提高命令行效率的重要工具，熟练使用可以大幅简化批量文件操作。</font>
