在 CentOS 系统中，用户管理是系统管理的重要部分，以下是常用的用户创建、修改等相关命令

## 创建用户

```bash
useradd [选项] 用户名
```

常用选项：

- `-m`：自动创建用户的家目录
- `-s`：指定用户的默认 shell，如`/bin/bash`
- `-g`：指定用户所属的初始组
- `-G`：指定用户所属的附加组

示例：创建一个带家目录且使用 bash 的用户 test

```bash
useradd -m -s /bin/bash test
```

## 设置 / 修改用户密码

```bash
passwd 用户名
```

执行后会提示输入新密码，输入时不会显示明文。

## 修改用户信息

```bash
usermod [选项] 用户名
```

常用选项：

- `-l`：修改用户名
- `-d`：修改用户家目录
- `-s`：修改默认 shell
- `-g`：修改用户所属的主要组
- `-G`：修改用户所属的附加组（会覆盖原有附加组，若要添加需加`-a`）

示例：将用户 test 的用户名改为 newtest

```bash
usermod -l newtest test
```

## 删除用户

```bash
userdel [选项] 用户名
```

常用选项：

- `-r`：删除用户的同时删除其家目录和邮件池

示例：彻底删除用户 test 及其家目录

```bash
userdel -rf test
```

## 查看用户信息

```bash
id 用户名
```

该命令会显示用户的 UID、GID 以及所属的组信息。

## 切换用户

```bash
su - 用户名
```

加上`-`会切换到该用户的环境，不加则保持当前环境。

## 临时赋予用户 root 权限执行命令

```bash
sudo 命令
```

需要用户在 sudoers 文件中有相应权限配置。

这些命令都需要管理员权限（通常通过`root`用户或`sudo`执行），使用时需谨慎，避免误操作影响系统安全和稳定性。
