<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">在 Linux 系统中，系统工作命令用于监控和管理系统状态、进程、资源等，以下是常用的系统工作命令：</font>

### <font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">1. 查看系统信息</font>
+ `**<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">uname</font>**`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">：显示系统内核信息</font>**<font style="color:rgb(249, 250, 251);background-color:rgb(252, 252, 252);">bash</font>**

```bash
uname -a  # 显示完整系统信息（内核版本、主机名、硬件架构等）
uname -r  # 仅显示内核版本
```

+ `**<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">hostname</font>**`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">：查看或设置主机名</font>**<font style="color:rgb(249, 250, 251);background-color:rgb(252, 252, 252);">bash</font>**

```bash
hostname          # 显示当前主机名
hostname newname  # 临时设置主机名（重启失效）
```

+ `**<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">cat /etc/os-release</font>**`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">：查看操作系统版本信息</font>**<font style="color:rgb(249, 250, 251);background-color:rgb(252, 252, 252);">bash</font>**

```bash
cat /etc/os-release  # 显示发行版名称、版本等（适用于主流Linux发行版）
```

### <font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">2. 进程管理</font>
+ `**<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">ps</font>**`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">：查看当前进程状态</font>**<font style="color:rgb(249, 250, 251);background-color:rgb(252, 252, 252);">bash</font>**

```bash
ps aux       # 显示所有进程（详细信息，包括用户、CPU占用等）
ps -ef       # 以全格式显示进程（包括父进程ID）
ps aux | grep "nginx"  # 过滤出包含nginx的进程
```

+ `**<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">top</font>**`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">：实时监控进程资源占用（动态更新）</font>**<font style="color:rgb(249, 250, 251);background-color:rgb(252, 252, 252);">bash</font>**

```bash
top  # 进入实时监控界面，按 q 退出
# 常用快捷键：P（按CPU排序）、M（按内存排序）、k（终止进程）
```

+ `**<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">htop</font>**`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">：</font>`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">top</font>`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);"> </font><font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">的增强版（需安装，界面更友好）</font>**<font style="color:rgb(249, 250, 251);background-color:rgb(252, 252, 252);">bash</font>**

```bash
htop  # 支持鼠标操作、颜色区分，功能更丰富
```

+ `**<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">kill</font>**`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">：终止进程</font>**<font style="color:rgb(249, 250, 251);background-color:rgb(252, 252, 252);">bash</font>**

```bash
kill 进程ID        # 发送终止信号（默认15）给进程
kill -9 进程ID     # 强制终止进程（信号9，无法忽略）
killall 进程名     # 终止所有同名进程（如 killall nginx）
```

### <font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">3. 资源监控</font>
+ `**<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">free</font>**`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">：查看内存使用情况</font>**<font style="color:rgb(249, 250, 251);background-color:rgb(252, 252, 252);">bash</font>**

```bash
free -h  # 以人类可读单位（G、M）显示内存和交换分区使用情况
```

+ `**<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">df</font>**`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">：查看磁盘空间使用情况</font>**<font style="color:rgb(249, 250, 251);background-color:rgb(252, 252, 252);">bash</font>**

```bash
df -h    # 显示所有文件系统的磁盘使用情况（含挂载点）
df -i    # 显示inode使用情况（避免因inode耗尽导致无法创建文件）
```

+ `**<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">du</font>**`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">：查看目录 / 文件的磁盘占用</font>**<font style="color:rgb(249, 250, 251);background-color:rgb(252, 252, 252);">bash</font>**

```bash
du -sh 目录名  # 显示目录总大小（-s：汇总，-h：友好单位）
du -h --max-depth=1  # 显示当前目录下一级子目录的大小
```

+ `**<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">vmstat</font>**`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">：监控系统整体资源（CPU、内存、IO 等）</font>**<font style="color:rgb(249, 250, 251);background-color:rgb(252, 252, 252);">bash</font>**

```bash
vmstat 2 5  # 每2秒输出一次，共输出5次（查看系统瓶颈常用）
```

+ `**<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">iostat</font>**`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">：监控磁盘 IO 性能（需安装</font><font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);"> </font>`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">sysstat</font>`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);"> </font><font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">包）</font>**<font style="color:rgb(249, 250, 251);background-color:rgb(252, 252, 252);">bash</font>**

```bash
iostat -x 1  # 每1秒显示一次磁盘IO详细统计（-x：扩展信息）
```

### <font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">4. 网络管理</font>
+ `**<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">ifconfig</font>**`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">：查看或配置网络接口（传统命令）</font>**<font style="color:rgb(249, 250, 251);background-color:rgb(252, 252, 252);">bash</font>**

```bash
ifconfig        # 显示所有网络接口信息（IP、MAC等）
ifconfig eth0   # 只显示eth0接口信息
```

+ `**<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">ip</font>**`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">：新一代网络管理命令（替代</font><font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);"> </font>`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">ifconfig</font>`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">）</font>**<font style="color:rgb(249, 250, 251);background-color:rgb(252, 252, 252);">bash</font>**

```bash
ip addr show       # 显示所有网络接口的IP地址
ip link set eth0 up/down  # 启用/禁用eth0接口
```

+ `**<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">ping</font>**`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">：测试网络连通性</font>**<font style="color:rgb(249, 250, 251);background-color:rgb(252, 252, 252);">bash</font>**

```bash
ping www.baidu.com  # 向目标发送ICMP包，按Ctrl+C停止
ping -c 4 192.168.1.1  # 只发送4个包
```

+ `**<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">netstat</font>**`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">：查看网络连接、端口监听等（常用</font><font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);"> </font>`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">-tuln</font>`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">）</font>**<font style="color:rgb(249, 250, 251);background-color:rgb(252, 252, 252);">bash</font>**

```bash
netstat -tuln  # 显示所有监听的TCP/UDP端口（t：TCP，u：UDP，l：监听，n：数字显示）
netstat -an    # 显示所有网络连接（包括已建立的连接）
```

+ `**<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">ss</font>**`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">：</font>`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">netstat</font>`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);"> </font><font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">的替代工具（更快、功能更多）</font>**<font style="color:rgb(249, 250, 251);background-color:rgb(252, 252, 252);">bash</font>**

```bash
ss -tuln  # 与 netstat -tuln 功能相同，但效率更高
```

### <font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">5. 系统负载与时间</font>
+ `**<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">uptime</font>**`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">：显示系统运行时间和负载</font>**<font style="color:rgb(249, 250, 251);background-color:rgb(252, 252, 252);">bash</font>**

```bash
uptime  # 输出格式：当前时间 运行时间 登录用户数 1/5/15分钟平均负载
```

+ `**<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">w</font>**`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">：显示当前登录用户及他们的活动</font>**<font style="color:rgb(249, 250, 251);background-color:rgb(252, 252, 252);">bash</font>**

```bash
w  # 显示用户登录时间、运行的命令等
```

+ `**<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">date</font>**`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">：查看或设置系统时间</font>**<font style="color:rgb(249, 250, 251);background-color:rgb(252, 252, 252);">bash</font>**

```bash
date                # 显示当前时间
date "+%Y-%m-%d %H:%M:%S"  # 自定义时间格式（年-月-日 时:分:秒）
```

+ `**<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">timedatectl</font>**`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">：管理系统时间和时区（systemd 系统）</font>**<font style="color:rgb(249, 250, 251);background-color:rgb(252, 252, 252);">bash</font>**

```bash
timedatectl               # 显示当前时间、时区等信息
timedatectl set-timezone Asia/Shanghai  # 设置时区为上海
```

### <font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">6. 其他实用命令</font>
+ `**<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">whoami</font>**`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">：显示当前登录用户名</font>**<font style="color:rgb(249, 250, 251);background-color:rgb(252, 252, 252);">bash</font>**

```bash
whoami  # 快速查看当前用户身份
```

+ `**<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">last</font>**`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">：查看用户登录历史</font>**<font style="color:rgb(249, 250, 251);background-color:rgb(252, 252, 252);">bash</font>**

```bash
last  # 显示最近的用户登录记录（包括登录时间、IP等）
```

+ `**<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">dmesg</font>**`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">：查看系统启动日志（硬件相关信息）</font>**<font style="color:rgb(249, 250, 251);background-color:rgb(252, 252, 252);">bash</font>**

```bash
dmesg | less  # 查看内核启动过程中的信息（如硬件检测、驱动加载等）
```

+ `**<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">systemctl</font>**`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">：管理系统服务（systemd 系统，替代</font><font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);"> </font>`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">service</font>`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">）</font>**<font style="color:rgb(249, 250, 251);background-color:rgb(252, 252, 252);">bash</font>**

```bash
systemctl status nginx  # 查看nginx服务状态
systemctl start nginx   # 启动服务
systemctl enable nginx  # 设置服务开机自启
```
