<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">在 CentOS 系统中，用户组管理与用户管理紧密相关，以下是常用的用户组管理命令：</font>

1. **<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">创建用户组</font>****<font style="color:rgb(249, 250, 251);background-color:rgb(252, 252, 252);">bash</font>**

```bash
groupadd [选项] 组名
```

<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">常用选项：</font>

    - `<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">-g</font>`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">：指定组 ID（GID），不指定则自动分配</font>

<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">示例：创建一个名为 develop 的用户组</font>**<font style="color:rgb(249, 250, 251);background-color:rgb(252, 252, 252);">bash</font>**

```bash
groupadd develop
```

2. **<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">修改用户组信息</font>****<font style="color:rgb(249, 250, 251);background-color:rgb(252, 252, 252);">bash</font>**

```bash
groupmod [选项] 组名
```

<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">常用选项：</font>

    - `<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">-n</font>`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">：修改组名</font>
    - `<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">-g</font>`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">：修改组 ID（GID）</font>

<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">示例：将组名 develop 改为 dev</font>**<font style="color:rgb(249, 250, 251);background-color:rgb(252, 252, 252);">bash</font>**

```bash
groupmod -n dev develop
```

3. **<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">删除用户组</font>****<font style="color:rgb(249, 250, 251);background-color:rgb(252, 252, 252);">bash</font>**

```bash
groupdel 组名
```

<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">注意：被删除的组不能是任何用户的主组，否则会删除失败。</font><font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">示例：删除名为 testgroup 的用户组</font>**<font style="color:rgb(249, 250, 251);background-color:rgb(252, 252, 252);">bash</font>**

```bash
groupdel testgroup
```

4. **<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">将用户添加到指定组</font>****<font style="color:rgb(249, 250, 251);background-color:rgb(252, 252, 252);">bash</font>**

```bash
usermod -aG 组名 用户名
```

<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">选项说明：</font>

    - `<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">-a</font>`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">：表示追加（append），避免覆盖用户原有的附加组</font>
    - `<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">-G</font>`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">：指定要添加到的附加组</font>

<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">示例：将用户 john 添加到 dev 组</font>**<font style="color:rgb(249, 250, 251);background-color:rgb(252, 252, 252);">bash</font>**

```bash
usermod -aG dev john
```

5. **<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">查看用户所属组</font>****<font style="color:rgb(249, 250, 251);background-color:rgb(252, 252, 252);">bash</font>**

```bash
groups 用户名
```

<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">示例：查看用户 john 所属的组</font>**<font style="color:rgb(249, 250, 251);background-color:rgb(252, 252, 252);">bash</font>**

```bash
groups john
```

6. **<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">查看系统中所有组</font>****<font style="color:rgb(249, 250, 251);background-color:rgb(252, 252, 252);">bash</font>**

```bash
cat /etc/group
```

<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">该文件存储了系统中所有组的信息，格式为：</font>`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">组名:密码:GID:成员列表</font>`

7. **<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">更改用户的主组</font>****<font style="color:rgb(249, 250, 251);background-color:rgb(252, 252, 252);">bash</font>**

```bash
usermod -g 新主组名 用户名
```

<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">示例：将用户 john 的主组改为 dev</font>**<font style="color:rgb(249, 250, 251);background-color:rgb(252, 252, 252);">bash</font>**

```bash
usermod -g dev john
```

<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">这些命令同样需要管理员权限执行，合理的用户组管理可以简化权限控制，提高系统安全性。例如，将多个用户添加到同一个组中，可以方便地对该组分配文件或目录的访问权限。</font>
