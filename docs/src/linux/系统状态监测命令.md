<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">在 Linux 系统中，系统状态监测命令用于实时查看或定期查看系统的运行状态、资源占用、进程负载情况等，是系统运维和问题排查的重要工具。以下是常用的系统状态监测命令：</font>

### <font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">一、进程与资源监控</font>
#### <font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">1.</font><font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);"> </font>`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">top</font>`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);"> </font><font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">- 实时进程监控（动态更新）</font>
<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">最常用的系统监控工具，可查看进程的 CPU、内存占用等信息。</font>

**<font style="color:rgb(249, 250, 251);background-color:rgb(252, 252, 252);">bash</font>**

```bash
top  # 进入交互界面，按 q 退出
```

**<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">常用快捷键</font>**<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">：</font>

+ `<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">P</font>`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">：按 CPU 使用率排序（默认）</font>
+ `<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">M</font>`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">：按内存使用率排序</font>
+ `<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">T</font>`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">：按进程运行时间排序</font>
+ `<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">k</font>`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">：输入进程 ID 终止指定进程</font>
+ `<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">u</font>`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">：查看指定用户的进程</font>
+ `<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">1</font>`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">：显示所有 CPU 核心的负载情况</font>

**<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">输出解读</font>**<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">：</font>

+ <font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">顶部显示系统整体信息（当前时间、运行时间、登录用户数、1/5/15 分钟负载）</font>
+ <font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">进程列表字段：PID（进程 ID）、USER（所属用户）、% CPU、% MEM、VSZ（虚拟内存）、RSS（物理内存）、STAT（状态）、COMMAND（命令）</font>

#### <font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">2.</font><font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);"> </font>`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">htop</font>`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);"> </font><font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">- 增强版进程监控（需安装）</font>
`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">top</font>`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);"> </font><font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">的替代工具，界面更友好，支持鼠标操作和颜色区分。</font>

**<font style="color:rgb(249, 250, 251);background-color:rgb(252, 252, 252);">bash</font>**

```bash
htop  # 进入交互界面，功能与 top 类似但更直观
```

#### <font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">3.</font><font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);"> </font>`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">ps</font>`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);"> </font><font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">- 静态进程查看（快照）</font>
<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">用于查看特定时间点的进程状态，常与管道结合过滤结果。</font>

**<font style="color:rgb(249, 250, 251);background-color:rgb(252, 252, 252);">bash</font>**

```bash
ps aux       # 显示所有进程的详细信息（包括用户、CPU、内存等）
ps -ef       # 以全格式显示进程（含父进程ID）
ps aux | grep "nginx"  # 筛选包含 nginx 的进程
```

### <font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">二、CPU 状态监测</font>
#### <font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">1.</font><font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);"> </font>`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">vmstat</font>`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);"> </font><font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">- 系统整体资源监控</font>
<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">显示 CPU、内存、IO 等系统整体状态，适合观察系统瓶颈。</font>

**<font style="color:rgb(249, 250, 251);background-color:rgb(252, 252, 252);">bash</font>**

```bash
vmstat 2 5  # 每2秒输出一次，共输出5次
```

**<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">关键字段</font>**<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">：</font>

+ `<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">us</font>`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">：用户空间程序占用 CPU 百分比</font>
+ `<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">sy</font>`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">：内核空间占用 CPU 百分比</font>
+ `<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">id</font>`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">：CPU 空闲百分比</font>
+ `<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">wa</font>`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">：等待 IO 的 CPU 百分比（数值过高可能表示磁盘 IO 瓶颈）</font>

#### <font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">2.</font><font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);"> </font>`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">mpstat</font>`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);"> </font><font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">- CPU 核心详细监控（需安装</font><font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);"> </font>`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">sysstat</font>`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">）</font>
<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">查看每个 CPU 核心的使用率，定位是否存在单核心负载过高问题。</font>

**<font style="color:rgb(249, 250, 251);background-color:rgb(252, 252, 252);">bash</font>**

```bash
mpstat -P ALL 1  # 每1秒显示所有CPU核心的状态
```

### <font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">三、内存状态监测</font>
#### <font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">1.</font><font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);"> </font>`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">free</font>`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);"> </font><font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">- 内存使用概况</font>
<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">显示物理内存、交换分区（swap）的使用情况。</font>

**<font style="color:rgb(249, 250, 251);background-color:rgb(252, 252, 252);">bash</font>**

```bash
free -h  # 以人类可读单位（G、M）显示
```

**<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">输出解读</font>**<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">：</font>

+ `<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">total</font>`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">：总内存</font>
+ `<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">used</font>`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">：已使用内存（包含缓存和缓冲区）</font>
+ `<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">free</font>`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">：完全空闲内存</font>
+ `<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">available</font>`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">：可用内存（实际可分配给新进程的内存）</font>
+ `<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">Swap</font>`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">：交换分区使用情况（频繁使用可能表示物理内存不足）</font>

#### <font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">2.</font><font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);"> </font>`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">vmstat</font>`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);"> </font><font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">- 内存交换监控</font>
<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">通过</font><font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);"> </font>`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">vmstat</font>`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);"> </font><font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">可观察内存页交换情况：</font>

**<font style="color:rgb(249, 250, 251);background-color:rgb(252, 252, 252);">bash</font>**

```bash
vmstat 1
# 关注 si（从swap读入内存的速率）和 so（从内存写入swap的速率），数值过高表示内存紧张
```

### <font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">四、磁盘 IO 监测</font>
#### <font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">1.</font><font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);"> </font>`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">iostat</font>`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);"> </font><font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">- 磁盘 IO 性能监控（需安装</font><font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);"> </font>`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">sysstat</font>`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">）</font>
<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">查看磁盘读写速率、IO 等待时间等，定位磁盘瓶颈。</font>

**<font style="color:rgb(249, 250, 251);background-color:rgb(252, 252, 252);">bash</font>**

```bash
iostat -x 1  # 每1秒显示一次扩展信息（包含IO细节）
```

**<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">关键字段</font>**<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">：</font>

+ `<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">%util</font>`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">：磁盘利用率（接近 100% 表示磁盘繁忙）</font>
+ `<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">r/s</font>`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">/</font>`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">w/s</font>`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">：每秒读写次数</font>
+ `<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">rkB/s</font>`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">/</font>`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">wkB/s</font>`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">：每秒读写数据量（KB）</font>
+ `<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">await</font>`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">：IO 请求平均等待时间（毫秒，数值过高表示 IO 压力大）</font>

#### <font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">2.</font><font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);"> </font>`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">iotop</font>`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);"> </font><font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">- 进程级磁盘 IO 监控（需安装）</font>
<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">查看每个进程的磁盘读写情况，定位消耗 IO 的进程。</font>

**<font style="color:rgb(249, 250, 251);background-color:rgb(252, 252, 252);">bash</font>**

```bash
iotop  # 类似 top 的交互界面，显示进程的 IO 占用
```
