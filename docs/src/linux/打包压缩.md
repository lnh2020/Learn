<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">在 Linux 系统中，打包和压缩文件是常用操作，用于节省存储空间和方便文件传输。常用的工具包括 </font>`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">tar</font>`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">（打包）、</font>`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">gzip</font>`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">、</font>`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">bzip2</font>`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">、</font>`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">xz</font>`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">（压缩）等，以下是具体用法：</font>

### <font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">一、</font>`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">tar</font>`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);"> </font><font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">命令（打包 / 解包）</font>
`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">tar</font>`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);"> </font><font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">本身仅用于打包（将多个文件 / 目录合并为一个文件），通常与压缩工具结合使用。</font>

#### <font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">1. 打包并压缩</font>
<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">格式：</font>`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">tar [选项] 压缩包名 源文件/目录</font>`

<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">常用选项：</font>

+ `<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">-c</font>`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">：创建打包文件</font>
+ `<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">-x</font>`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">：解包</font>
+ `<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">-v</font>`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">：显示过程（verbose）</font>
+ `<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">-f</font>`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">：指定打包文件名（必须放在最后）</font>
+ `<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">-z</font>`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">：用 gzip 压缩 / 解压（.tar.gz 或 .tgz）</font>
+ `<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">-j</font>`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">：用 bzip2 压缩 / 解压（.tar.bz2）</font>
+ `<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">-J</font>`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">：用 xz 压缩 / 解压（.tar.xz）</font>
+ `<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">-C</font>`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">：指定解压目录</font>

**<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">示例</font>**<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">：</font>

**<font style="color:rgb(249, 250, 251);background-color:rgb(252, 252, 252);">bash</font>**

```bash
# 用gzip压缩（最常用）
tar -zcvf docs.tar.gz /home/docs  # 将/home/docs目录压缩为docs.tar.gz

# 用bzip2压缩（压缩率略高，速度较慢）
tar -jcvf images.tar.bz2 *.jpg    # 将所有jpg文件压缩为images.tar.bz2

# 用xz压缩（压缩率最高，速度最慢）
tar -Jcvf data.tar.xz /var/data   # 将/var/data目录压缩为data.tar.xz
```

#### <font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">2. 解包并解压</font>
<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">格式：</font>`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">tar [选项] 压缩包名</font>`

**<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">示例</font>**<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">：</font>

**<font style="color:rgb(249, 250, 251);background-color:rgb(252, 252, 252);">bash</font>**

```bash
# 解压tar.gz文件到当前目录
tar -zxvf docs.tar.gz

# 解压tar.bz2文件到指定目录/tmp
tar -jxvf images.tar.bz2 -C /tmp

# 解压tar.xz文件（不显示过程）
tar -Jxf data.tar.xz
```

### <font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">二、单一文件压缩工具</font>
<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">适用于压缩单个文件（不能直接压缩目录）。</font>

#### <font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">1.</font><font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);"> </font>`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">gzip</font>`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">（.gz）</font>
**<font style="color:rgb(249, 250, 251);background-color:rgb(252, 252, 252);">bash</font>**

```bash
gzip 文件名          # 压缩文件（原文件会被删除，生成文件名.gz）
gzip -d 文件名.gz    # 解压（-d表示decompress）
gzip -k 文件名       # 压缩并保留原文件（-k：keep）
```

<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">示例：</font>

**<font style="color:rgb(249, 250, 251);background-color:rgb(252, 252, 252);">bash</font>**

```bash
gzip report.txt      # 压缩为report.txt.gz，原文件删除
gzip -dk logs.txt.gz # 解压logs.txt.gz并保留压缩包
```

#### <font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">2.</font><font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);"> </font>`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">bzip2</font>`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">（.bz2）</font>
<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">压缩率高于 gzip，但速度较慢：</font>

**<font style="color:rgb(249, 250, 251);background-color:rgb(252, 252, 252);">bash</font>**

```bash
bzip2 文件名         # 压缩文件
bzip2 -d 文件名.bz2  # 解压
bzip2 -k 文件名      # 压缩并保留原文件
```

#### <font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">3.</font><font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);"> </font>`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">xz</font>`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">（.xz）</font>
<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">压缩率最高，速度最慢：</font>

**<font style="color:rgb(249, 250, 251);background-color:rgb(252, 252, 252);">bash</font>**

```bash
xz 文件名            # 压缩文件
xz -d 文件名.xz      # 解压
xz -k 文件名         # 压缩并保留原文件
```

### <font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">三、</font>`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">zip</font>`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);"> </font><font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">和</font><font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);"> </font>`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">unzip</font>`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">（跨平台兼容）</font>
`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">zip</font>`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);"> </font><font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">格式在 Windows 和 Linux 间兼容性好，支持压缩目录。</font>

#### <font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">1. 安装（部分系统默认不安装）</font>
**<font style="color:rgb(249, 250, 251);background-color:rgb(252, 252, 252);">bash</font>**

```bash
# CentOS/RHEL
sudo yum install zip unzip

# Ubuntu/Debian
sudo apt install zip unzip
```

#### <font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">2. 使用方法</font>
**<font style="color:rgb(249, 250, 251);background-color:rgb(252, 252, 252);">bash</font>**

```bash
# 压缩文件/目录为zip包
zip -r archive.zip dir1 file1.txt  # -r：递归压缩目录

# 解压zip包到当前目录
unzip archive.zip

# 解压到指定目录
unzip archive.zip -d /target/dir

# 查看zip包内容（不解压）
unzip -l archive.zip
```

### <font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">注意事项</font>
+ <font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">压缩目录时，</font>`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">tar</font>`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);"> </font><font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">必须加</font><font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);"> </font>`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">-r</font>`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);"> </font><font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">选项（递归），</font>`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">zip</font>`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);"> </font><font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">也需要</font><font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);"> </font>`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">-r</font>`
+ <font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">解压时若要指定目录，</font>`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">tar</font>`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);"> </font><font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">用</font><font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);"> </font>`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">-C</font>`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">，</font>`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">unzip</font>`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);"> </font><font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">用</font><font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);"> </font>`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">-d</font>`
+ <font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">压缩率与速度通常成反比，根据需求选择合适格式</font>

<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">通过这些工具，可以高效地管理文件的打包与压缩，节省存储空间并简化文件传输流程。</font>
