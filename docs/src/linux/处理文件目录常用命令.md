<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">在 Linux 系统中，处理文件和目录的命令是日常操作的基础，以下是常用命令及示例：</font>

`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">pwd</font>`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">：查看当前路径	</font>

### <font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">1. 查看目录内容</font>
**<font style="color:rgb(249, 250, 251);background-color:rgb(252, 252, 252);">bash</font>**

```bash
ls [选项] [目录]
```

+ <font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">常用选项：</font>
    - `<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">-l</font>`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">：详细列表形式显示（包含权限、大小等信息）</font>
    - `<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">-a</font>`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">：显示所有文件（包括隐藏文件，以</font><font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);"> </font>`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">.</font>`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);"> </font><font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">开头）</font>
    - `<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">-h</font>`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">：以人类可读的单位显示大小（如 K、M）</font>
    - `<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">-R</font>`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">：递归显示子目录内容</font>

<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">示例：</font>

**<font style="color:rgb(249, 250, 251);background-color:rgb(252, 252, 252);">bash</font>**

```bash
ls -lha /home  # 详细显示/home目录所有文件（含隐藏）并以友好单位显示大小
```

### <font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">2. 切换目录</font>
**<font style="color:rgb(249, 250, 251);background-color:rgb(252, 252, 252);">bash</font>**

```bash
cd [目录路径]
```

+ <font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">特殊用法：</font>
    - `<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">cd</font>`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);"> </font><font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">或</font><font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);"> </font>`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">cd ~</font>`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">：切换到当前用户的家目录</font>
    - `<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">cd ..</font>`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">：切换到上级目录</font>
    - `<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">cd -</font>`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">：切换到上一次所在目录</font>

<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">示例：</font>

**<font style="color:rgb(249, 250, 251);background-color:rgb(252, 252, 252);">bash</font>**

```bash
cd /var/log  # 切换到/var/log目录
cd ../tmp    # 从当前目录切换到上级目录的tmp子目录
```

### <font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">3. 创建目录</font>
**<font style="color:rgb(249, 250, 251);background-color:rgb(252, 252, 252);">bash</font>**

```bash
mkdir [选项] 目录名
```

+ <font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">常用选项：</font>
    - `<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">-p</font>`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">：递归创建多级目录（父目录不存在时自动创建）</font>

<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">示例：</font>

**<font style="color:rgb(249, 250, 251);background-color:rgb(252, 252, 252);">bash</font>**

```bash
mkdir docs                  # 创建单个目录docs
mkdir -p project/{src,lib}  # 同时创建project目录及其子目录src和lib
```

### <font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">4. 创建空文件</font>
**<font style="color:rgb(249, 250, 251);background-color:rgb(252, 252, 252);">bash</font>**

```bash
touch 文件名
```

+ <font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">若文件不存在则创建空文件，若已存在则更新其修改时间</font>

<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">示例：</font>

**<font style="color:rgb(249, 250, 251);background-color:rgb(252, 252, 252);">bash</font>**

```bash
touch note.txt          # 创建空文件note.txt
touch logs/{a,b}.log    # 同时创建logs目录下的a.log和b.log
```

### <font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">5. 复制文件 / 目录</font>
**<font style="color:rgb(249, 250, 251);background-color:rgb(252, 252, 252);">bash</font>**

```bash
cp [选项] 源文件/目录 目标路径
```

+ <font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">常用选项：</font>
    - `<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">-r</font>`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);"> </font><font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">或</font><font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);"> </font>`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">-R</font>`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">：递归复制目录（必须用于目录复制）</font>
    - `<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">-i</font>`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">：覆盖前提示确认</font>
    - `<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">-v</font>`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">：显示复制过程</font>

<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">示例：</font>

**<font style="color:rgb(249, 250, 251);background-color:rgb(252, 252, 252);">bash</font>**

```bash
cp file.txt /backup              # 复制file.txt到/backup目录
cp -r docs/ /tmp                 # 递归复制docs目录到/tmp
cp -iv *.pdf ~/documents         # 复制所有pdf文件到documents目录，覆盖前提示
```

### <font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">6. 移动 / 重命名文件 / 目录</font>
**<font style="color:rgb(249, 250, 251);background-color:rgb(252, 252, 252);">bash</font>**

```bash
mv [选项] 源文件/目录 目标路径
```

+ <font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">功能：移动文件 / 目录，或在同一目录下重命名</font>
+ <font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">常用选项：</font>`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">-i</font>`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">（覆盖前提示）、</font>`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">-v</font>`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">（显示移动过程）</font>

<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">示例：</font>

**<font style="color:rgb(249, 250, 251);background-color:rgb(252, 252, 252);">bash</font>**

```bash
mv old.txt new.txt               # 重命名文件
mv report.pdf ~/downloads        # 移动文件到downloads目录
mv -i data/ /archive             # 移动目录到archive，覆盖前提示
```

### <font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">7. 删除文件 / 目录</font>
**<font style="color:rgb(249, 250, 251);background-color:rgb(252, 252, 252);">bash</font>**

```bash
rm [选项] 文件/目录
```

+ <font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">常用选项：</font>
    - `<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">-r</font>`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);"> </font><font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">或</font><font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);"> </font>`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">-R</font>`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">：递归删除目录（必须用于目录删除）</font>
    - `<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">-f</font>`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">：强制删除（不提示确认）</font>
    - `<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">-i</font>`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">：删除前提示确认（默认对重要文件生效）</font>

<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">示例：</font>

**<font style="color:rgb(249, 250, 251);background-color:rgb(252, 252, 252);">bash</font>**

```bash
rm temp.txt              # 删除单个文件（可能提示确认）
rm -rf old_dir/          # 强制递归删除目录及其内容（慎用！）
rm -i *.tmp              # 删除所有tmp文件，每个都提示确认
```

### <font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">8. 查看文件内容</font>
+ `<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">cat 文件名</font>`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">：一次性显示整个文件内容（适合短文件）</font>
+ `<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">more 文件名</font>`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">：分页显示（按空格键翻页，q 退出）</font>
+ `<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">less 文件名</font>`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">：更强大的分页查看（支持上下滚动、搜索）</font>
+ `<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">head -n 行数 文件名</font>`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">：显示文件前 N 行（默认前 10 行）</font>
+ `<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">tail -n 行数 文件名</font>`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">：显示文件后 N 行（默认后 10 行）</font>
+ `<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">tail -f 文件名</font>`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">：实时跟踪文件新增内容（常用于日志监控）</font>

<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">示例：</font>

**<font style="color:rgb(249, 250, 251);background-color:rgb(252, 252, 252);">bash</font>**

```bash
cat config.ini                  # 查看配置文件
less /var/log/messages          # 分页查看系统日志
tail -n 20 access.log           # 查看访问日志最后20行
tail -f /var/log/nginx/error.log # 实时监控nginx错误日志
```

### <font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">9. 搜索文件 / 目录</font>
**<font style="color:rgb(249, 250, 251);background-color:rgb(252, 252, 252);">bash</font>**

```bash
find 搜索路径 [选项] 条件
```

<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">示例：</font>

**<font style="color:rgb(249, 250, 251);background-color:rgb(252, 252, 252);">bash</font>**

```bash
find /home -name "*.txt"        # 在/home下搜索所有txt文件
find /var -type d -name "log*"  # 在/var下搜索以log开头的目录
find / -size +100M              # 在根目录下搜索大于100M的文件
```

### <font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">10. 查看当前目录路径</font>
**<font style="color:rgb(249, 250, 251);background-color:rgb(252, 252, 252);">bash</font>**

```bash
pwd
```

+ <font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">显示当前工作目录的绝对路径</font>

<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">这些命令是 Linux 文件系统操作的核心，熟练掌握后能极大提高工作效率。使用时注意：删除操作（</font>`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">rm -rf</font>`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">）需格外谨慎，避免误删重要文件；处理目录时记得添加 </font>`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);">-r</font>`<font style="color:rgb(0, 0, 0);background-color:rgb(252, 252, 252);"> 选项。</font>
